📌 Project Overview

This project focuses on analyzing sales data to identify trends, patterns, and business insights that can help improve decision-making. By performing data cleaning, exploration, and visualization, the analysis highlights key factors affecting sales performance such as product demand, regional sales distribution, and monthly revenue trends.

🎯 Objectives

Analyze overall sales performance

Identify top-selling products

Understand customer purchasing patterns

Analyze regional and category-wise sales

Discover seasonal sales trends

Provide insights to support business strategies

📂 Dataset

The dataset contains information such as:

Order ID

Product Name

Category

Sales Amount

Quantity

Customer Details

Region / Location

Order Date
